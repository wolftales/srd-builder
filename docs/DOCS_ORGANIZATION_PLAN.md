## 6. Benefits

**Organization:**
- Clear distinction: reference (root) vs historical (releases) vs future (planning)
- Bug investigations archived but discoverable
- Release history in chronological order

**Discoverability:**
- New contributors find living docs in root
- Historical context in releases/
- Planning docs separated from completed work

**Maintenance:**
- Easier to update living docs (fewer files in root)
- Historical docs frozen (no accidental updates)
- Clear pattern for future releases
